## 介绍

通过组件及通用功能封装，尽量抹平各端差异，然后按规范套路去实现模块业务逻辑

## 特性

- 多语言
- 多主题
- 自定义图标
- 组件化
- 分包预加载
- 登录跳转控制
- 操作权限控制
- vue3
